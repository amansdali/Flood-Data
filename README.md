# Flood Data
Hackathon Demo Project for NewHacks 2024.
Flood Data is a crowdsourcing tool for collecting real-time data about floods in flood-prone areas. Features a simple, intuitive UI, and secure password-protected accounts, allowing users to report information about floods to help with research and information gathering.
